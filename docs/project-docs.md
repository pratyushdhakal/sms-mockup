# 🍎 SMS Project Blueprint

> Minimalist, Apple-inspired linear design. High-contrast. Clean.

---

### ⚙️ Engine
- **Dev:** `npm run dev`
- **Build:** `npm run build` (Typecheck + Vite)
- **Lint:** `npm run lint`

### 🏗️ Stack
- **Core:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS (@tailwindcss/vite)
- **Motion:** `framer-motion`
- **Integrations:** `@gianpieropuleo/radix-mcp-server`, `tailwindcss-mcp-server`

### 🎨 Design System
- **Aesthetic:** Minimalist, linear, high-contrast, ample whitespace.
- **Locale:** Nepalese context (Location, Phone) in English (e.g., *Morang, 9800980260*).
- **Icons:** Consistent, lightweight SVGs only.

### 🛠️ Conventions
- **Components:** Functional, descriptive naming.
- **Typing:** Strict `typescript-eslint` compliance.
- **Interaction:** Fluid `framer-motion` transitions.
